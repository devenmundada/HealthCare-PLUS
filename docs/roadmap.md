# Roadmap

## Current (v1.0 — In Development)

Working:
- Hospital search and map (50+ real Indian hospitals)
- Doctor directory (15+ specialists)
- AI chat assistant (Ollama + Phi-3 Mini)
- JWT authentication
- Dark/light mode

In progress:
- Appointment booking complete flow
- Health metrics dashboard
- Emergency response button

## Phase 2 — Next 3 Months

- Telemedicine: video consultation via WebRTC
- Digital health records with encryption
- Medicine delivery integration with local pharmacies
- Ayushman Bharat and PM-JAY scheme lookup

## Phase 3 — Next 6 Months

- Medical image analysis (skin conditions, X-ray preliminary review)
- Hindi and regional language UI (Tamil, Telugu, Bengali)
- Voice interface for accessibility
- Advanced symptom checker

## Phase 4 — Next 12 Months

- React Native mobile app (iOS + Android)
- 500+ hospitals across all Indian states
- 500+ verified doctors
- Cloud-hosted AI option (no local Ollama required)
- Corporate health plans (B2B)

## Not Planned (Explicit Non-Goals)

- Replacing licensed medical professionals
- Storing diagnostic images without explicit user consent
- Selling user health data
